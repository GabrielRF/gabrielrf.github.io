Telegram

Este aplicativo tornou-se essencial no meu dia-a-dia. Tão essencial que passei a estudar suas APIs como hobby.

Primeiro foram bots. Bots em todos os lugares. Bots que podiam fazer de tudo. Imediamente fiquei inspirado e motivado a estudar o tema.

Bots do Telegram

O resultado desse estudo é mostrado aqui. Atualmente mantenho três bots em funcionamento ininterrupto. Eles são rápidos, confiáveis e capazes de responder diversas requisições por dia.

  • lmgtfy_bot



    Primeira tentativa de criar um bot usando a API oficial do Telegram e uma API em Python, dispoível no GitHub


    https://github.com/eternnoir/pyTelegramBotAPI


    Esse bot transforma qualquer texto recebido pelo usuário em um link do site "Let Me Google That For You".


    Clique aqui para testar.

  • RastreioBot



    Bot do Telegram que rastreia pacotes dos Correios.


    O usuário digita o código de rastreamento e o bot responde a última localização do pacote disponível no sistema.


    Clique aqui para testar.

  • Safe_bot



    Bot que cria senhas e calcula hashes.


    O usuário é guiado durante todo o processo por opções que aparecem no local do teclado.


    Tem código aberto e não mantém registros. Disponível no GitHub


    Clique aqui para testar.

  • UrlProBot



    O UrlProBot é capaz de encurtar URLs longas e de expandir as curtas.


    Seu uso é bastante simples. Basta que o usuário envie uma URL e o bot se encarrega do resto.


    Clique aqui para testar.

Canais do Telegram

Alguns meses depois o Telegram disponibilizou os canais. Um canal é capaz de enviar mensagens à um número ilimitado de usuários. Então, comecei a procurar por algo cujo público fosse o maior possível. A resposta foi: Economizar dinheiro.

  • PromoApps



    Meu primeiro canal totalmente controlado por um bot. Coleta as informações enviadas por usuários de uma comunidade, por isso o canal é rápido e com informações bem selecionadas.


    A proposta do canal é permitir que os usuários possam aproveitar as promoções o mais rápido possível enquanto os aplicativos estão mais baratos que o normal. Tais promoções normalmente duram um dia.


    Clique aqui para testar.


    Já testou? Avalie o canal!

  • PromoPassagens



    Canal totalmente automático. Coleta informações de sites especializados em encontrar descontos em passagens aéreas e as compartilha.


    A vantagem deste canal é notificar as pessoas quando uma passagem barata está disponível. Normalmente há poucos lugares dispoíveis, por isso, saber rapidamente pode ser um diferencial.


    Clique aqui para testar.


    Já testou? Avalie o canal!

  • On_Sale



    Mais um canal totalmente automático. Funciona de maneira similiar ao PromoPassagens. A diferença é não haver filtros sobre o tipo de promoção. Qualquer ítem que se torna mais barato pode aparecer no canal.


    Os usuários podem encontrar desde consoles de jogos até liquidificadores. Não há filtros. O canal foi feito para quem gosta de acompanhar preços baixos.


    Clique aqui para testar.


    Já testou? Avalie o canal!

  • BsbDF



    Este canal é uma boa fonte de informações para moradores de Brasília. Reúne informações de diversas fontes e compartilha somente o que é relevante.


    Automaticamente notifica os usuários em tempo real sobre trânsito, falhas de enerngia, greves, manutenções planejadas ou emergenciais e notícias do governo.


    Clique aqui para testar.


    Já testou? Avalie o canal!
